# 🧩 LaklyConvert

**Image to Braille ASCII Art Converter**

Превращайте любые изображения в текстовый арт с использованием символов шрифта Брайля.  
Мощный, гибкий и быстрый конвертер с множеством настроек для идеального результата.

---

## ✨ Особенности

- **Высокое качество** – оптимальный подбор Braille-символов по MSE (включая взвешенный MSE), автосдвиг сетки, адаптивный порог, постобработка.
- **Гибкие настройки** – регулируйте контраст, яркость, порог, метод бинаризации (Otsu, адаптивный Брэдли, дитеринг Флойда–Стейнберга или Байера).
- **Интуитивный интерфейс** – тёмная тема, Drag&Drop, копирование в буфер, сохранение в файл, быстрый запуск.
- **Сохранение настроек** – все параметры автоматически запоминаются в `config.json`.
- **Кроссплатформенность** – работает на Windows, Linux и macOS (при наличии Python).

## 🚀 Быстрый старт

### 1. Установка зависимостей

Просто запустите **`Setup.bat`** – он автоматически установит все необходимые библиотеки (numpy, Pillow, а также опциональные scipy и tkinterdnd2).

Либо вручную:

```bash
pip install numpy pillow scipy tkinterdnd2
```

### 2. Запуск

Дважды кликните по **`Ready.bat`**.  
При первом запуске программа проверит зависимости и предложит создать файл `.skip_check` для мгновенного запуска в будущем.

Или запустите напрямую:

```bash
python Main.py
```

## 🧰 Основные возможности

| Функция | Описание |
|---------|----------|
| **Открытие изображения** | Поддерживаются все популярные форматы (PNG, JPG, BMP, GIF, TIFF, WEBP). |
| **Настройка ширины** | Изменяйте количество символов по горизонтали (от 2 до 500). |
| **Вертикальный масштаб** | Компенсируйте сжатие/растяжение арта. |
| **Контраст и яркость** | Тонкая настройка перед бинаризацией. |
| **Метод бинаризации** | Otsu, адаптивный порог (Брэдли), дитеринг Флойда или Байера. |
| **Подбор символа** | `direct` (быстро), `MSE` (точно), `Weighted MSE` (ещё точнее с весами). |
| **Автосдвиг** | Автоматически выравнивает сетку для лучшего качества линий. |
| **Взвешенное усреднение** | Сглаживает пиксели для более естественного вида. |
| **Постобработка** | Удаляет одиночные шумовые пиксели. |
| **Плотный режим** | Заполняет пустые блоки точкой (U+2804) для сплошного арта. |
| **Тёмная тема** | Комфортная работа в любых условиях. |
| **Drag & Drop** | Перетащите файл прямо в окно программы. |
| **Копирование в буфер** | Один клик – и арт готов к вставке. |
| **Сохранение в файл** | Экспорт в `.txt` с UTF-8. |
| **Сохранение настроек** | Все параметры запоминаются в `config.json`. |

## 🛠 Управление

- **Открыть** – выбрать изображение.
- **Применить** – переконвертировать с текущими настройками.
- **Копировать** – скопировать арт в буфер обмена.
- **Сохранить** – сохранить в текстовый файл.
- **Очистить** – сбросить текстовое поле.
- **Сбросить настройки** – вернуть значения по умолчанию.

Настройки сгруппированы в три строки для удобства.
